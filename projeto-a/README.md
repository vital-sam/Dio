# 📄 Projeto A — Sistema de Inteligência Comercial e Logística com IA

## 🧠 Visão Geral
Sistema automatizado para consolidar dados de vendas, gerar insights e tomar decisões com IA.

## 🎯 Objetivos
- Consolidar dados de múltiplas fontes
- Gerar insights acionáveis
- Identificar produtos populares por país
- Otimizar logística
- Automatizar decisões

## 🏗️ Estrutura
```
app/
models/
config/
data/
run.py
requirements.txt
README.md
```

## 🚀 Como executar

```bash
git clone https://github.com/vital-sam/Dio.git
cd Dio/projeto-a
pip install -r requirements.txt
python run.py
```

## ⚙️ Pipeline
- Extração de dados CSV
- Limpeza e normalização
- Geração de insights
- Decisão automatizada

## 🤖 IA aplicada
- Previsão de demanda (Linear Regression)
- Motor de decisão automatizado

## 📊 Saída
- Insights de vendas
- Ranking por país
- Ações sugeridas

## 🔄 Evoluções futuras
- Integração com APIs
- Banco de dados
- Dashboard (BI)
- Deploy em cloud
